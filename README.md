# Digital-Persona
Based on https://github.com/AtlasFoundation/AvatarCreator

## How to run
* git clone https://github.com/Artificial-Leap/Digital-Persona
* cd Digital-Persona/Client
* npm install
* npm run dev
* cd ../server
* npm install
* npm start
